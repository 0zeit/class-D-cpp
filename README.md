# Phase 1: C++ 기초 및 SFML 입문
> 목표: C++ 기초 문법 습득, SFML로 뭔가 돌아가는것 제작

- SFML 기본(Window, Shape, Event)
- 헤더 및 소스 파일 해설
- 포인터 기본

2. Week 2-3(8h): Pong 게임(충돌, 점수, 게임 루프 이해)
3. Week 4-5(8h): 간단한 그림판(그리기, 지우기)
4. Week 6(4h)  : 자유롭게 뭔가 만들어보기 및 숙제
